# Courier Shipping Calculator 
The program `courier.py` is designed to assist a courier company in calculating the cost of sending a parcel. It provides a user-friendly interface to input the price of the package and the total distance of the delivery in km. The program then determines the final cost of the package by considering various delivery preferences.

The program is written in python.

## Usage
1. Clone the repository to your local machine or download the 'courier.py' file directly.

2. Open an IDE of your choice and navigate to the folder containing 'courier.py'.

3. Run the program.
